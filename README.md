# TodoList

## Требования:

1. [макет](https://www.figma.com/design/FI7Zr50cE6kradGFIZYRSP/Simple-ToDo-List-Design--Community-?node-id=1-2&t=QYXk6HhytGsJCmz9-0)

4. Если список тасок пустой, пользователь должен видеть это на экране.
